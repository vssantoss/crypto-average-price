**Detailed Calculation Plan: Non-Stablecoin USD And BRL Cost**

**Summary**
Add USD cost-basis tracking for non-stablecoins and calculate BRL acquisition cost for stablecoin-funded buys using the linked stablecoin’s own BRL average cost, not PTAX. Linked trade fees are included when they affect the bought coin or spent stablecoin; third-coin fees remain separate dispositions.

**New Fields**
- `USD Tx Cost`: USD/stablecoin amount attached to the row.
- `USD Balance`: remaining USD cost basis after the row.
- `USD Avg Price`: `USD Balance / Running Balance`.
- `BRL Cost Rate`: actual BRL-per-stablecoin rate used for automatic BRL cost.
- Existing `BRL Tx Cost`: gains a stablecoin-average-cost automatic source for linked non-stablecoin buys.

**Fee Handling**
- Same-instrument linked fees continue to use:

```text
Net Tx Quantity = Tx Quantity - same-instrument linked Trade Fee
```

- If the fee is in the bought non-stablecoin:

```text
net acquired quantity = bought coin Tx Quantity - bought coin fee
USD Avg Price = USD Tx Cost / net acquired quantity
BRL Avg Price = BRL Tx Cost / net acquired quantity
```

- If the fee is in the spent stablecoin:

```text
USD Tx Cost = abs(stablecoin Tx Quantity - stablecoin fee)
BRL Tx Cost = USD Tx Cost * stablecoin BRL Avg Price before sale
```

- If the fee is in a third coin:
  - Do not add it to the bought asset’s USD or BRL cost.
  - Treat it as a separate disposition of the fee coin, using existing fee/disposition cost-basis behavior.

**USD Tx Cost**
- For a non-stablecoin `BUY` linked to a USD/stablecoin `SELL`:

```text
USD Tx Cost = abs(linked stablecoin Net Tx Quantity)
```

- If linked stablecoin net quantity is unavailable:

```text
USD Tx Cost = abs(linked stablecoin Transaction Cost)
```

- If the user manually enters USD cost:

```text
USD Tx Cost = manual USD Tx Cost
```

- For non-stablecoin deposits, use only manual USD cost.
- For stablecoin rows, keep the new non-stablecoin USD cost columns blank in the table.

**USD Balance**
- Remaining USD cost basis for the current non-stablecoin holding after the row.

For buys or deposits with known USD cost:

```text
USD Balance after = USD Balance before + USD Tx Cost
```

For sells, withdrawals, same-coin fees, or dust:

```text
USD Avg Price before = USD Balance before / Balance before
USD cost removed = USD Avg Price before * abs(Net Tx Quantity)
USD Balance after = USD Balance before - USD cost removed
```

For soft stake rewards:

```text
USD Balance after = USD Balance before
```

**USD Avg Price**
- Calculated as:

```text
USD Avg Price = USD Balance / Running Balance
```

- Blank when `USD Balance` is unknown or running balance is zero/negative.
- If the user enters a manual USD average-price seed:

```text
USD Balance at seed row = manual USD Avg Price * Running Balance
```

- USD seeds propagate forward and backward independently from BRL seeds.

**Stablecoin BRL Avg Price Before Trade**
- For the linked stablecoin `SELL`, read the stablecoin BRL average price before applying the sale row:

```text
stablecoin BRL Avg Price before = stablecoin BRL Balance before / stablecoin Balance before
```

- This pre-sale value is required because the stablecoin balance may become zero after the trade.
- If this value is missing or invalid, the app cannot automatically calculate the non-stablecoin BRL cost.

**BRL Cost Rate**
- For a non-stablecoin `BUY` funded by a linked stablecoin `SELL`:

```text
BRL Cost Rate = linked stablecoin BRL Avg Price before sale
```

- This is not PTAX.
- Blank when no stablecoin average cost exists or when BRL cost is purely manual.

**BRL Tx Cost**
- For a non-stablecoin `BUY` linked to a stablecoin `SELL`:

```text
BRL Tx Cost = USD Tx Cost * BRL Cost Rate
```

Expanded with stablecoin fee handling:

```text
BRL Tx Cost = abs(linked stablecoin Net Tx Quantity) * linked stablecoin BRL Avg Price before sale
```

- Manual BRL Tx Cost wins over automatic BRL cost.
- PTAX is not used as fallback for this acquisition path.

**BRL Balance And Avg Price**
- Existing formulas continue, but BRL balance can now receive automatic stablecoin-derived BRL cost.

For buys/deposits with known BRL cost:

```text
BRL Balance after = BRL Balance before + BRL Tx Cost
```

For sells, withdrawals, same-coin fees, or dust:

```text
BRL Avg Price before = BRL Balance before / Balance before
BRL cost removed = BRL Avg Price before * abs(Net Tx Quantity)
BRL Balance after = BRL Balance before - BRL cost removed
```

```text
BRL Avg Price = BRL Balance / Running Balance
```

**Example**
Stablecoin state before trade:

```text
USDC Balance before = 1,001
USDC BRL Balance before = R$ 5,205.20
USDC BRL Avg Price before = R$ 5.20
```

Trade:

```text
SELL 1,000 USDC
TRADE_FEE 1 USDC
BUY 0.02 BTC
```

Calculated BTC values:

```text
USD Tx Cost = 1,001
BRL Cost Rate = R$ 5.20
BRL Tx Cost = 1,001 * 5.20 = R$ 5,205.20
BTC USD Balance = $1,001
BTC USD Avg Price = 1,001 / 0.02 = $50,050
BTC BRL Balance = R$ 5,205.20
BTC BRL Avg Price = 5,205.20 / 0.02 = R$ 260,260
```

If the fee is instead `0.0001 BTC`:

```text
net acquired BTC = 0.0199
USD Tx Cost = 1,000
BRL Tx Cost = 1,000 * 5.20 = R$ 5,200
BTC USD Avg Price = 1,000 / 0.0199
BTC BRL Avg Price = 5,200 / 0.0199
```

- Verify bought-coin fees reduce acquired quantity and increase average price.
- Verify stablecoin fees increase USD/BRL cost transferred to the bought coin.
- Verify third-coin fees remain separate dispositions.
- Verify missing stablecoin BRL average cost leaves automatic BRL Tx Cost blank.
- Verify manual USD and BRL overrides take priority.
